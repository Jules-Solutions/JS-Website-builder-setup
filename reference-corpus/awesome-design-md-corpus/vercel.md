# Vercel — DESIGN.md

**Personality:** Stark, geometric, high-contrast. Pure black-and-white minimalism with surgical precision. Confident enough to use almost no color — the triangle and the typography carry it.

## Color
- Pure monochrome: black `#000000` and white `#FFFFFF` as the dominant pair.
- Grays: a precise neutral ramp for borders/secondary text (`#888`, `#666`, `#333`, `#171717`).
- Color appears only as functional accent or in occasional sharp gradients (e.g. blue/pink) on feature pages — rationed hard.
- **Take-away:** confidence through absence of color; contrast is the design.

## Typography
- **Geist** — Vercel's own sans + mono (Geist Sans / Geist Mono). Geometric, clean, slightly technical.
- Strong contrast: large tight headlines vs. calm body; mono for code + technical labels.

## Layout & spacing
- Crisp grid, generous whitespace, perfect alignment. High information clarity.
- Small-to-medium radii (~6–8px); hairline borders; sharp, deliberate edges.

## Components
- Buttons: black-on-white / white-on-black, minimal, fast. Secondary = bordered ghost.
- Cards: white (or near-black in dark), fine border, tight padding, often with a code block.
- Signature: the black triangle; flawless dark mode; deploy-preview UI polish.

## Motion
- Minimal, fast, exact. Subtle fades + crisp hovers. Nothing decorative.

## Voice (microcopy)
- Concise, technical, declarative. Developer-to-developer.

## Signature moves
- Black/white + Geist + geometric precision + hairline borders + the triangle = "Vercel."

## When to reference this
Briefs wanting **minimal, stark, geometric, high-contrast, technical** — dev infrastructure, modern portfolios, agencies, anything that signals confidence through restraint.
