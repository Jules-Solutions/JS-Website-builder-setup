# Stripe — DESIGN.md

**Personality:** Precise, premium, developer-respecting. The gold standard for "fintech that engineers trust." Clean white canvas punctuated by bold, optimistic gradients.

## Color
- Brand: "blurple" — `#635BFF` (the signature Stripe purple). Anchors the whole identity.
- Canvas: crisp white `#FFFFFF` + very light cool grays for sections.
- Text: deep slate/ink — `#0A2540` (Stripe's dark navy) for headings; muted slate for body.
- Accents: vivid multi-stop **gradients** (cyan→purple→pink→orange) used in heroes + section dividers.
- **Take-away:** restrained white base lets the blurple + gradient do all the talking.

## Typography
- Sans-serif, modern grotesque (Stripe uses "Sohne"/"Söhne"). Tight, confident headings; highly legible body.
- Strong size contrast between big marketing headlines and calm body copy.

## Layout & spacing
- Generous, grid-disciplined, lots of whitespace. Angled/skewed gradient sections are a signature.
- Medium radii (~8px) on cards + inputs. Crisp, never soft.

## Components
- Buttons: blurple fill, white label, subtle depth; clear hover. Famous for snappy, polished micro-states.
- Cards: white, fine border, soft shadow, precise padding. Code samples rendered beautifully (developer audience).
- Signature: animated angled gradient hero; immaculate API docs styling.

## Motion
- Refined and purposeful — smooth gradient animation, crisp hover transitions. Polished, never gratuitous.

## Voice (microcopy)
- Clear, confident, technically credible. Speaks to developers and operators as peers.

## Signature moves
- Blurple `#635BFF` + white canvas + angled multi-color gradients + flawless docs = "Stripe."

## When to reference this
Briefs wanting **precise, premium, trustworthy, developer-credible** — fintech, B2B SaaS, API products, anything that must look polished and engineered.
