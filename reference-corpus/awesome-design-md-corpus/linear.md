# Linear — DESIGN.md

**Personality:** Fast, precise, dark, keyboard-first. The defining "modern dev-tool" aesthetic — near-black surfaces, an indigo glow, and obsessive micro-interaction craft. Everything feels instantaneous.

## Color
- Canvas: near-black, slightly blue — `#08090A` / `#101113`.
- Accent: indigo/violet — `#5E6AD2` (Linear purple). Subtle, premium, never neon.
- Text: high-contrast off-white `#F7F8F8`; muted gray `#8A8F98` for secondary.
- Surfaces: layered dark grays with hairline borders (`~#23252A`); faint purple glows behind heroes.
- **Take-away:** dark-first with *one* restrained indigo accent and very low-chroma neutrals.

## Typography
- Inter (or Inter-like grotesque), tight tracking on headings. Crisp, technical, legible.
- Modest size contrast — hierarchy comes from weight + color, not size theatrics.

## Layout & spacing
- Dense but breathable; tight, consistent spacing; sharp grid alignment.
- Small radii (~6–8px). Hairline borders everywhere. Subtle gradients + glow, never flat-boring.

## Components
- Buttons: subtle — low-contrast fills, crisp borders, fast hover. Quiet until needed.
- Cards/rows: dark surface, hairline border, tight padding; keyboard-focus states are first-class.
- Signature: command-menu (Cmd-K), buttery transitions, gradient mesh hero glow.

## Motion
- Fast (120–200ms), precise, physics-aware. Micro-interactions are the brand's craft signal.

## Voice (microcopy)
- Terse, confident, builder-to-builder. No fluff.

## Signature moves
- `#08090A` canvas + `#5E6AD2` indigo + hairline borders + purple glow + Cmd-K = "Linear."

## When to reference this
Briefs wanting **fast, precise, dark, premium, technical** — dev tools, modern B2B SaaS, productivity apps, anything that should feel engineered and instantaneous.
