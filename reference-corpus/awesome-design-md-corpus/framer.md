# Framer — DESIGN.md

**Personality:** Bold, modern, motion-led. A site-builder that shows off — vivid gradients, big confident type, and animation as a first-class design element. "Design in motion."

## Color
- Canvas: high-contrast — deep black `#0A0A0A` and clean white, switched per section.
- Accent: vivid blue/indigo `#0099FF`-ish + gradient blends (blue→purple→pink). Bright, energetic.
- Text: white on dark / near-black on light; crisp grays for secondary.
- **Take-away:** high-contrast base + vivid gradient accents + motion = energetic-modern.

## Typography
- Sans-serif, bold geometric (Inter / their own). Large, tight, confident headlines; clean body.
- Heavy weight + size contrast; type often animated on entrance.

## Layout & spacing
- Spacious, bold sections; gradient panels; generous whitespace punctuated by motion.
- Medium radii (~10–12px); smooth, modern. Interactive demos embedded.

## Components
- Buttons: bold gradient/solid fills, rounded, animated hover. Confident and lively.
- Cards: clean surface, soft shadow, often with motion-on-scroll reveals.
- Signature: scroll-triggered animation, gradient meshes, springy interactions, live previews.

## Motion
- **The brand.** Spring physics, scroll-choreography, entrance animations everywhere — purposeful and smooth, showcasing what the tool can do. (Still honors reduce-motion.)

## Voice (microcopy)
- Bold, confident, creator-focused. "Ship sites that move." Energetic and modern.

## Signature moves
- High-contrast canvas + vivid gradients + bold type + signature spring motion = "Framer."

## When to reference this
Briefs wanting **bold, modern, motion-rich, energetic** — creative agencies, product launches, portfolios, anything where animation and a contemporary edge are part of the pitch.
