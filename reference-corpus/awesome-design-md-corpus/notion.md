# Notion — DESIGN.md

**Personality:** Friendly, editorial, approachable-minimal. Mostly black-and-white with warm off-white surfaces, hand-drawn line illustrations, and emoji as a playful accent. Reads like a clean, inviting document.

## Color
- Canvas: warm off-white `#FFFFFF` / `#F7F6F3` (Notion's faint warm paper).
- Text: near-black `#37352F` (warm-tinted, not pure black) + muted grays for secondary.
- Accents: a muted, desaturated palette (the "Notion colors": soft blue, brown, orange, green, pink) used as tags/highlights — never loud.
- Emoji as functional color/iconography.
- **Take-away:** monochrome base + warm-tinted ink + muted tag colors + emoji = friendly, not corporate.

## Typography
- Display: a refined serif (Notion uses "Lyon") for marketing headlines — editorial warmth.
- UI/body: clean sans (Inter-like / system). Comfortable, document-like reading.

## Layout & spacing
- Generous whitespace; block-based, document-like structure; clear hierarchy.
- Medium radii (~6–8px) on blocks/cards; soft, friendly. Hand-drawn line illustrations as a signature.

## Components
- Buttons: understated; black primary, quiet secondary. Friendly, not flashy.
- Cards/blocks: off-white surface, light border, comfortable padding, emoji icon.
- Signature: the block model, emoji headers, hand-drawn illustrations, gallery/board views.

## Motion
- Gentle and minimal — soft hovers, smooth block drag. Calm.

## Voice (microcopy)
- Warm, plain, encouraging, lightly playful. Human and helpful.

## Signature moves
- Warm off-white + warm-ink black + emoji + serif headlines + hand-drawn illustration = "Notion."

## When to reference this
Briefs wanting **friendly, editorial, approachable, document-like** — productivity tools, knowledge bases, education, content-first products that should feel welcoming.
