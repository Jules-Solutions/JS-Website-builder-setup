# Raycast — DESIGN.md

**Personality:** Sleek, dark, glassy, premium-native. A macOS-feeling command launcher — deep dark surfaces, vivid red-coral accent, glassmorphism, and rich gradients. Polished and a little futuristic.

## Color
- Brand: Raycast red-coral — `#FF6363`. Vivid, warm, energetic accent.
- Canvas: deep near-black — `#0D0D0D` / `#171717` with glassy translucent panels.
- Text: white `#FFFFFF` + muted grays.
- Accents: rich multi-stop gradients (coral→purple→blue) on heroes + extension art.
- **Take-away:** dark glass + coral accent + vivid gradients = sleek, premium, modern.

## Typography
- Sans-serif, clean modern (Inter-like). Tight confident headlines; crisp body.
- Mono for shortcuts/keys (the command-palette heritage). Keyboard hints styled as kbd chips.

## Layout & spacing
- Focused, centered, app-like; glassy floating panels; generous dark whitespace.
- Medium radii (~10–12px); soft glass blur; subtle inner glows + gradients.

## Components
- Buttons: coral primary, dark glass secondary, rounded. Crisp hover; keyboard-first affordances.
- Cards: translucent dark glass, soft border, gradient accents, kbd chips.
- Signature: command-palette UI, glassmorphism, vivid gradient hero, keyboard-shortcut chips.

## Motion
- Smooth, fast, refined — glassy reveals, gradient shifts, snappy keyboard interactions.

## Voice (microcopy)
- Sharp, confident, power-user-focused. Efficient and a little playful.

## Signature moves
- Dark glass + coral `#FF6363` + vivid gradients + command-palette + kbd chips = "Raycast."

## When to reference this
Briefs wanting **sleek, dark, glassy, premium-modern** — power-user tools, launchers, AI/productivity apps, anything that should feel polished, fast, and a bit futuristic.
